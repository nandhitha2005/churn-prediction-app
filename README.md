# Customer Churn Prediction System

This project is an end-to-end Machine Learning application that predicts whether a telecom customer is likely to churn. It analyzes customer demographics, service usage, and billing details to estimate churn probability and classify customers into risk categories.

The system is designed to help businesses identify at-risk customers and take proactive retention actions. It provides not only predictions but also business-oriented insights such as risk levels and recommended actions.

The model is built using a Random Forest algorithm and deployed using Gradio on Hugging Face Spaces, making it accessible through an interactive web interface.

This project predicts whether a telecom customer is likely to churn using a Machine Learning model.

## 🔍 Features
- Predicts churn risk (High / Moderate / Low)
- Provides business-level recommendations
- Interactive web interface using Gradio

## 🧠 Model
- Algorithm: Random Forest
- Accuracy: ~79%
- Input: Customer demographics & service usage
- Output: Churn probability + risk classification

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Gradio
